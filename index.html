Notable Projects

Enhancing VPN Performance using an Adaptive Protocol Selector System (APSS)

MSc. Dissertation Project

This dissertation evaluated the performance of OpenVPN, IPSec, and WireGuard across public, niche, and private networks. It highlighted OpenVPN's suitability for public networks due to strong security features, WireGuard's superior performance in private networks, and IPSec's balance of security and performance for niche corporate or educational networks. The research also involved the conceptual design of an Adaptive Protocol Selector System (APSS) to dynamically select optimal VPN protocols based on real-time network conditions (SSID Analysis, Network Security Assessment, Geolocation Data) and user preferences, aiming to enhance network security and efficiency without user intervention.
